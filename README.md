# Interactive Quiz App

A responsive 3-level quiz application built with HTML, CSS, and Vanilla JavaScript. Test your knowledge across Easy, Medium, and Hard difficulty levels with a timer and instant score feedback.


## ✨ Features
- **3 Difficulty Levels** - Easy, Medium, and Hard with different question sets
- **Timer System** - 30 seconds per question with auto-submit on timeout  
- **Instant Feedback** - Correct and wrong answers are highlighted immediately
- **Score Tracking** - Real-time score updates and final result summary
- **Responsive Design** - Works smoothly on desktop, tablet, and mobile
- **No Dependencies** - Built with pure HTML, CSS, and JavaScript

## 🛠️ Tech Stack
- **HTML5** - Structure and semantic markup
- **CSS3** - Styling, animations, and responsive layout
- **JavaScript (ES6)** - DOM manipulation, event handling, and game logic
